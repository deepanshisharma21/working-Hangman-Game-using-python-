# 🎮 Hangman Game

A colorful terminal-based Hangman game built using Python.

## Features

- 🎨 Colored terminal output
- ❤️ Lives counter
- 📋 Guessed letters tracker
- 🔄 Play Again option
- 🎉 Win/Lose messages
- 🎲 Random word generation

## Technologies Used

- Python
- Colorama

## How to Run

```bash
pip install -r requirements.txt
python main.py
